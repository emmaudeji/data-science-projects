## Method 3 — Age Group Analysis (More Business-Friendly)

Analysts rarely stop at raw age.

We create segments.

---

Code.

```python
df["Age Group"] = pd.cut(
    df["Age"],
    bins=[18,30,45,60,70],
    labels=[
        "18-30",
        "31-45",
        "46-60",
        "60+"
    ]
)
```

---

### What is `pd.cut()`?

Feature engineering.

Converts:

```txt id="kq8gv7"
23

44

57
```

into:

```txt id="fr5jz0"
18–30

31–45

46–60
```

---

Now analyze.

---

Code.

```python
df.groupby(
"Age Group"
)["Purchase Amount (USD)"]\
.mean()
```

---

Visualize.

```python
sns.boxplot(
    data=df,
    x="Age Group",
    y="Purchase Amount (USD)"
)
```

---

Business story example.

```txt id="07h9lr"
46–60 customers spend highest.

Opportunity:

premium campaigns for older buyers.
```
