# Better Visualization 3: Quadrant Analysis

Create categories.

```python
sales_median = product_analysis["Sales"].median()
revenue_median = product_analysis["Revenue"].median()
```

Then classify products.

```python
def classify(row):

    if row["Sales"] >= sales_median and row["Revenue"] >= revenue_median:
        return "Star"

    elif row["Sales"] >= sales_median:
        return "Popular"

    elif row["Revenue"] >= revenue_median:
        return "Premium"

    else:
        return "Weak"

product_analysis["Segment"] = (
    product_analysis.apply(
        classify,
        axis=1
    )
)
```

Now you can do:

```python
product_analysis.groupby(
    "Segment"
).size()
```

Result:

```txt
Star       8
Popular    5
Premium    4
Weak      10
```

Now you're speaking business language.
