## Cleaning Example

Suppose.

---

Before.

```python
df["Gender"].unique()
```

Result.

```txt id="q7cd7h"
['Male','male','Female']
```

---

Clean.

```python
df["Gender"] = (
    df["Gender"]
    .str.strip()
    .str.title()
)
```

---

Explanation.

---

### `.str.strip()`

Removes whitespace.

Example.

```txt id="0m0phd"
" Male "
→
"Male"
```

---

### `.str.title()`

Standardizes capitalization.

```txt id="t3fjjy"
male → Male
```

---

Very common cleaning technique.
