Here are the most useful **Jupyter Notebook shortcuts inside VS Code**.

### 1. Switch Modes

Jupyter notebooks have **2 modes**:

* **Command Mode** → manage cells
* **Edit Mode** → type code inside a cell

| Action                 | Shortcut |
| ---------------------- | -------- |
| Enter **Edit Mode**    | `Enter`  |
| Enter **Command Mode** | `Esc`    |

---

### 2. Cell Navigation

| Action                  | Shortcut               |
| ----------------------- | ---------------------- |
| Select cell above       | `↑`                    |
| Select cell below       | `↓`                    |
| Move cursor within code | Arrow keys (Edit Mode) |

---

### 3. Run Cells

| Action                              | Shortcut        |
| ----------------------------------- | --------------- |
| Run current cell                    | `Shift + Enter` |
| Run current cell & stay             | `Ctrl + Enter`  |
| Run current cell & insert new below | `Alt + Enter`   |

---

### 4. Cell Creation / Deletion *(Command Mode — press `Esc` first)*

| Action         | Shortcut                |
| -------------- | ----------------------- |
| Add cell below | `B`                     |
| Add cell above | `A`                     |
| Delete cell    | `D D` *(press D twice)* |
| Cut cell       | `X`                     |
| Copy cell      | `C`                     |
| Paste cell     | `V`                     |
| Paste above    | `Shift + V`             |

---

### 5. Change Cell Type

| Action        | Shortcut |
| ------------- | -------- |
| Code cell     | `Y`      |
| Markdown cell | `M`      |
| Raw cell      | `R`      |

---

### 6. Notebook Editing

| Action                   | Shortcut           |
| ------------------------ | ------------------ |
| Undo cell deletion       | `Z`                |
| Rename variable / symbol | `F2`               |
| Find in notebook         | `Ctrl + F`         |
| Command Palette          | `Ctrl + Shift + P` |

---

### 7. Code Productivity (VS Code goodies)

| Action                   | Shortcut          |
| ------------------------ | ----------------- |
| IntelliSense suggestions | `Ctrl + Space`    |
| Multi-cursor             | `Alt + Click`     |
| Comment line             | `Ctrl + /`        |
| Format document/cell     | `Shift + Alt + F` |
| Go to definition         | `F12`             |
| Peek definition          | `Alt + F12`       |

---

### 8. Kernel & Notebook Actions

| Action           | Shortcut             |
| ---------------- | -------------------- |
| Interrupt kernel | Use notebook toolbar |
| Restart kernel   | Use notebook toolbar |
| Run all cells    | Notebook toolbar     |
| Clear outputs    | Command Palette      |

---

### 9. My most-used workflow shortcuts (high ROI)

```txt
Esc       → command mode
Enter     → edit mode
Shift+Enter → run cell + next
A / B     → add cell above / below
DD        → delete cell
M / Y     → markdown / code
Z         → undo delete
Ctrl+/    → comment code
Ctrl+Space → autocomplete
```

A good mental model:

* **Writing code** → `Enter`
* **Managing notebook cells** → `Esc`
* **Executing work** → `Shift + Enter`
